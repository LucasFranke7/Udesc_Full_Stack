Select

O select, é um elemento que oferece um conjunto de opções que são visiveis ao clicarmos na setinha ao lado direito do elemento. Por padrao a primeira opção é adicionada, com tudo, é possivel oferecer uma opção pre definida usando o atributo -selected-. Por padrao esse elemento permite apenas uma escolha, com tudo é possivel definir a escolha de multiplas opções usando o atributo -multiple-



Fieldset

Apesar de não ser uma tag de entrada de daodos, o fieldset é frequentemente usado em formularios. Ele produz um efeito em boda que divide uma opção de dados de modo a facilitar a visualização agrupando elementos relacionados, quando a tag legend é utilizada é adicionada uma descrição a sua estrutura.
